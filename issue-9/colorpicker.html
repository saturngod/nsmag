Mac OSX Color Picker

color picker အသုံးပြုပုံ နည်းလမ်း အချို့
ရေးသားသူ : saturngod

Mac Color Picker ကို iOS developer တွေ သုံးကြမှာပါ။ အဲဒီ color picker အတွက် အသုံးဝင်တဲ့ နည်းလမ်းလေး အချို့ ဖော်ပြပေးလိုက်ပါတယ်။

##HexPicker

OS X မှာ clock picker ကို အသုံးပြုပြီးတော့ ကျွန်တော်တို့တွေ iOS development မှာ အရောင်တွေ ရွေးတာတွေကို ဖန်တီးပါတယ်။ သို့ပေမယ့် color picker မှာ hex value ထည့်လို့မရပါဘူး။

Hex Picker က color picker မှာ #RRGGBB value ကို ထည့်သွင်းလို့ ရအောင် ဖန်တီးပေးပါတယ်။

Using Color from Image

Color picker မှာ image ကို အသုံးပြုပြီးတော့ color select လုပ်လို့ရပါတယ်။

Image Tab ကို ပြောင်းလိုက်ပါ။

Image ကို drag & Drop ဆွဲချလိုက်ပါ။

ပြီးရင် image ကို လွယ်လင့်တကူ color select လုပ်လို့ရပါတယ်။

အဲဒါလေးကို အတော်ကို အသုံးဝင်ပါတယ်။ တကယ်လို့ photoshop ကို အသုံးပြုရင် photoshop က swatch panel ကို screenshot ရိုက်ပြီးတော့ ရလာတဲ့ image ကို drag and drop လုပ်လိုက်ပါ။

ဒါဆိုရင် ကျွန်တော်တို့တွေ photoshop က color swatch ကို iOS color picker မှာ လွယ်လင့် တကူ selct လုပ်လို့ရပါပြီ။
